
# my_smartapp

## Introduction

This project is a Flutter mobile application called **My SmartApp**.
The app is designed to help users track, analyze, and understand their health or sports performance.
It includes pages for injuries, match statistics, health analysis, and educational videos.
The goal of this version is to present the structure and interface of the system with an organized design and clear navigation.

---

## Objective

The main objective of this project is to build a complete and organized mobile app that supports users in monitoring their physical condition and performance.
It provides pages for data entry, visualization, and analysis related to health and sports activities.

---

## Project Overview

The app includes several pages, each one created for a specific function:

| File                                                                           | Description                                                                         |
| ------------------------------------------------------------------------------ | ----------------------------------------------------------------------------------- |
| `Login.dart`                                                                   | Login page for users to access the app.                                             |
| `matchstats.dart`, `Matchstatsdatem.dart`, `Matchstavidio.dart`                | Pages related to match statistics and performance videos.                           |
| `injuries.dart`, `Injuriesdeta.dart`, `Injurievidio.dart`                      | Pages that display information and videos about sports injuries.                    |
| `healthanaly.dart`, `biganaly.dart`, `healthyvideo.dart`, `healthydeterm.dart` | Pages for health and performance analysis with supporting videos.                   |
| `chat.dart`                                                                    | Chat page, prepared for future integration to help users communicate or get advice. |
| `analys.dart`, `newlogic.dart`                                                 | Pages containing logic or data processing functions.                                |
| `done.dart`, `delet.dart`, `nopass.dart`                                       | Pages for different result messages such as success, delete, or error.              |
| `langu.dart`                                                                   | Language selection page.                                                            |
| `main.dart`                                                                    | The main entry point of the application.                                            |
